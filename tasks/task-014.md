# task-014: 实现历史记录查询接口

## 描述
实现面试历史记录的列表查询和详情查看。

## 具体内容
- 实现 GET /api/interview/list（当前用户的所有面试会话，支持按 status/position 筛选，按 created_at 排序）
- 写列表查询测试用例（正常查询 / 按条件筛选 / 空结果）
- 实现 GET /api/interview/{id}/detail（面试详情，包含对话记录和评价）
- 写详情查询测试用例（正常查询 / 会话不存在 / 无评价的会话）
- 跑测试确认通过

## 验收标准
- pytest 测试通过
- 列表接口返回分页数据，支持筛选
- 详情接口返回完整对话 + 评价
