# task-007: 实现简历解析服务

## 描述
调 LLM 解析简历文件，提取结构化信息。

## 具体内容
- 实现简历解析服务（读取上传的 PDF/图片 → 调 LLM → 提取结构化简历文本）
- 设计简历解析 prompt（输入文件内容，输出个人信息/教育/工作经历/技能）
- 写解析服务测试用例（输入模拟简历文本，输出包含必填字段）
- 实现解析结果写入 Resume.parsed_content
- 跑测试确认通过

## 验收标准
- pytest 测试通过
- 上传简历后 parsed_content 有 AI 解析的结构化内容
