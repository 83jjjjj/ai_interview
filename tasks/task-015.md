# task-015: 前端评价展示 + 历史记录页面

## 描述
前端实现面试评价展示和历史记录列表。

## 具体内容
- 实现面试结束后的评价展示页面（总体评分 + 维度雷达图 + 改进建议列表）
- 实现历史记录列表页面（调用 /api/interview/list，显示会话列表）
- 实现面试详情页面（调用 /api/interview/{id}/detail，显示对话 + 评价）
- 集成 ECharts 或 Chart.js 展示维度评分

## 验收标准
- 面试结束后能看到评价页面，评分和建议正确展示
- 历史记录列表可点击进入详情
- 维度评分有可视化图表
