# task-004: 初始化前端项目 + 登录注册页面

## 描述
搭建 Vue 3 + Element Plus 前端项目，实现登录和注册页面。

## 具体内容
- 用 Vite 创建 Vue 3 项目
- 安装配置 Element Plus
- 配置前端开发代理（转发 /api 到后端）
- 实现注册页面（表单验证 + 调用 /api/register）
- 实现登录页面（表单验证 + 调用 /api/login + 存 token）
- 实现路由守卫（未登录跳转登录页）

## 验收标准
- `npm run dev` 能启动
- 浏览器能注册、能登录、登录后跳转到首页
- 未登录访问首页自动跳转到登录页
