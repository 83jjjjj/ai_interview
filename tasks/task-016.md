# task-016: 实现个人能力分析接口

## 描述
综合用户所有面试评价，生成能力分析和发展建议。

## 具体内容
- 实现 GET /api/analysis/{user_id}（查询该用户所有 Evaluation，调 LLM 生成趋势分析 + 发展建议）
- 设计分析 prompt（输入多轮评价的维度数据，输出能力趋势 + 发展建议，结构化 JSON）
- 写分析接口测试用例（正常查询 / 无面试记录）
- 跑测试确认通过

## 验收标准
- pytest 测试通过
- 有多次面试记录时返回趋势分析和发展建议
- 无记录时返回合理提示
